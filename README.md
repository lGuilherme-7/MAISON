# MAISON — Loja Virtual (Front-end)

> Projeto fictício de e-commerce de moda criado para fins de portfólio e demonstração de layout.

🔗 **Demo ao vivo:** [lguilherme-7.github.io/MAISON](https://lguilherme-7.github.io/MAISON/)

---

## Sobre

A **MAISON** é uma loja virtual fictícia desenvolvida com foco em design elegante e experiência mobile-first. Nenhum produto é real e nenhuma transação é processada — trata-se exclusivamente de uma demonstração de front-end.

## Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Home com banner, categorias e grid de produtos |
| `product.html` | Página de produto com galeria, cores e tamanhos |
| `cart.html` | Sacola de compras com cupom e frete |
| `checkout.html` | Checkout com formulário, pagamento e resumo |
| `desenvolvimento.html` | Aviso de site fictício / portfólio |

## Tecnologias

- HTML5
- CSS3 (Mobile First, sem frameworks)
- JavaScript puro (sem bibliotecas)
- LocalStorage (sacola e favoritos)
- GitHub Pages (hospedagem)

## Funcionalidades

- ✅ Navbar responsiva com mega menu (desktop) e accordion (mobile)
- ✅ Galeria de imagens no produto
- ✅ Seleção de cor e tamanho
- ✅ Sacola com localStorage
- ✅ Sistema de favoritos persistente
- ✅ Cupom de desconto simulado
- ✅ Cálculo de frete por CEP (simulado)
- ✅ Checkout com validação de formulário
- ✅ Preview do cartão de crédito em tempo real
- ✅ Modal de confirmação de pedido

---

> ⚠ **Aviso:** Este é um projeto fictício para fins de portfólio. Não há back-end, coleta de dados ou transações reais.
